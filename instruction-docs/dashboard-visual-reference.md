## Dark Theme Specification

All components are designed dark-first with these principles:

1. **Contrast Ratio:**
   - Text on background: â‰¥ 4.5:1
   - Large text (â‰¥18pt): â‰¥ 3:1

2. **Layering:**
   - Page: #0F172A (darkest)
   - Card: #1E293B (lighter)
   - Hover: #334155 (lightest)

3. **Borders:**
   - Subtle: 10% white opacity
   - Prominent: 20% white opacity
   - Focus: 50% primary color

4. **Glow Effects:**
   - Use low opacity (10-20%)
   - Large blur radius (48-96px)
   - Strategic placement (corners)
